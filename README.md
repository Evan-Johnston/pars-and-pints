🏌️ About the Project

Pars & Pints is a modern golf simulator and bar experience built around golf, drinks, events, and community.

This website serves as the main online home for customers to learn more about Pars & Pints, find important business information, book simulator time, view upcoming events, and stay connected.

The goal of this project is to create a site that feels clean, premium, fun, and easy to use while staying lightweight and simple to maintain.

⸻

✨ Features

* Fully responsive layout
* Modern landing page design
* Mobile-friendly navigation
* Clear call-to-action sections
* Simulator booking entry points
* Event and tournament-ready sections
* Business information and contact areas
* Clean visual structure
